# Бот модерации — Гражданское Общество

## Функции
1. ✅ Приветствие + проверка подписки на канал (5 мин → кик)
2. ↗️ Перемещение ссылок/файлов из Флудилки в ССЫЛКИ
3. 🚫 Автоматический бан ботов при добавлении
4. ⌨️ Автоисправление раскладки (ghbdtn → привет)
5. 💬 Оживление чата при тишине 1 час

## Деплой на Render.com
1. Залей код на GitHub
2. Подключи репозиторий в Render.com
3. Установи BOT_TOKEN в переменных окружения
4. Деплой запустится автоматически

## Переменные окружения
- BOT_TOKEN — токен бота (обязательно)
- GROUP_ID — ID группы (по умолчанию: -1001787318055)
- CHANNEL_ID — канал для проверки подписки (по умолчанию: @societycivil)
- FLOODIKA_THREAD — ID топика Флудилка (по умолчанию: 438392)
- LINKS_THREAD — ID топика Ссылки (по умолчанию: 460367)
- CHECK_DELAY — секунды до кика за неподписку (по умолчанию: 300)
- SILENCE_TIMEOUT — секунды тишины до оживления (по умолчанию: 3600)
